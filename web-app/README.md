# Web Application

## Tech Stack / Dependecies

- React
- React Router DOM
- Axios: for calling the graphql endpoint
- Typescript

## How to run the Application?

### 1. Install the dependencies

```Bash
$ npm i
```

### 2. Run the code

```Bash
$ npm run start
```
